# 買い物言い訳ツール

商品名と金額を入力すると、その買い物をもっともらしく正当化する短い言い訳をブラウザ内で生成するシンプルなWebアプリです。少しくだらないのに、つい何回か試したくなる雰囲気を目指しています。

## サービス概要

- 商品名と金額を入力すると、日本語の短い言い訳をランダム生成します
- 金額をもとに「100回使うと仮定すれば」「1年間使う前提なら」などの計算を含めます
- 商品名のキーワードから簡単なカテゴリー判定を行い、言い訳の方向性を少し変えます
- 生成後は「別の言い訳を考える」と「コピーする」が使えます

## 外部APIを使用していないこと

このアプリは以下の条件を満たしています。

- 外部APIを使用していません
- OpenAI APIなどの生成AI APIを使用していません
- APIキーや環境変数は不要です
- バックエンド、データベース、ログイン機能はありません
- 外部フォントやCDNを使用していません
- ブラウザ内だけで動作する静的サイトです

## ローカルでの起動方法

特別な準備は不要です。

1. このプロジェクトをダウンロードします
2. `index.html` をブラウザで開きます

ローカルサーバーを使いたい場合も、任意の静的ファイルサーバーでそのまま配信できます。

## GitHub Pagesで公開できること

このプロジェクトはHTML/CSS/JavaScriptだけで構成された静的サイトなので、GitHub Pagesなどの無料ホスティングでそのまま公開できます。

## ファイル構成

- `index.html` : 画面の構造
- `style.css` : 見た目とレスポンシブ対応
- `script.js` : 入力チェック、カテゴリー判定、言い訳生成、コピー機能
- `README.md` : この説明書

## 言い訳テンプレートを追加する方法

`script.js` の中にある配列を編集すると、言い訳のバリエーションを増やせます。

- `logicTemplates` に新しい文章テンプレート関数を追加する
- `categoryIntros` にカテゴリーごとの導入文を追加する
- `bridges` に接続表現を追加する
- `categoryRules` にキーワードを追加すると、商品名の判定精度を上げられます

テンプレート関数では以下の値が使えます。

- `item` : 商品名
- `formattedAmount` : 3桁区切りの金額
- `perUse100` : 100回使うと仮定した1回あたり金額
- `perDay1Year` : 1年間使う前提の1日あたり金額
- `perDay3Years` : 3年間使えた場合の1日あたり金額
- `compareAmount` : 比較用に少し大きめにした金額

文章は80〜160文字程度を目安に調整すると、読みやすさを保ちやすくなります。
