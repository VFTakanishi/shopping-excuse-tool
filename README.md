# 買い物言い訳ツール

商品名と金額を入力すると、その買い物を短くて少しくだらない言い訳に変換する、ブラウザ完結の静的Webアプリです。

長い説明ではなく、一瞬だけ「まあ、そうかも」と思える言い回しを目指しています。

## サービス概要

- 商品名と金額を入力すると、短い日本語の言い訳を生成します
- 同じ入力でも「別の言い訳を考える」で違う候補が出るようにしています
- 商品名のキーワードからカテゴリを判定し、カテゴリと金額帯ごとに文を切り替えます
- 入力エラーはその場で日本語表示します
- コピー機能付きです

## 外部APIを使用していないこと

このアプリは次の条件で動作します。

- 外部APIを使用しません
- OpenAI APIなどの生成AI APIを使用しません
- APIキーは不要です
- バックエンドを使用しません
- データベースを使用しません
- ログイン機能はありません
- 外部フォントやCDNを使用しません
- すべてブラウザ内だけで完結します

## ローカルでの起動方法

特別な準備は不要です。

1. このプロジェクトのファイル一式を手元に置きます
2. `index.html` をブラウザで開きます

ローカルサーバーなしでも基本機能が動作します。

## 言い訳テンプレートを追加する方法

言い訳の判定と文言は `script.js` の次の定義にまとまっています。

- `SUBTYPE_RULES`
  商品名に含まれる単語と、どのカテゴリ・商品タイプとして扱うかを定義します
- `PRICE_THRESHOLDS`
  カテゴリごとの金額帯を定義します
- `TEMPLATE_MAP`
  商品タイプごとの言い訳テンプレート本体です

### 追加手順

1. `SUBTYPE_RULES` に商品名のキーワードを追加します
2. 必要なら `PRICE_THRESHOLDS` に金額帯を追加または調整します
3. `TEMPLATE_MAP` に対象タイプの `low` `normal` `high` `extreme` の候補を追加します
4. すべて 1〜2 文、120 文字以内を目安に調整します

### 例

```js
{ subtype: "book", category: "genericProduct", keywords: ["本", "書籍", "参考書"] }
```

```js
book: {
  low: [
    ({ item, formattedAmount }) =>
      `${formattedAmount}の${item}なら、一冊で考え方が一つ増えれば十分です。`
  ],
  normal: [
    ({ item, formattedAmount }) =>
      `${item}に${formattedAmount}なら、暇つぶしより中身が残る方の出費です。`
  ],
  high: [
    ({ item, formattedAmount }) =>
      `${formattedAmount}の${item}は高めですが、何度か開くならまだ話せます。`
  ],
  extreme: [
    ({ item, formattedAmount }) =>
      `${formattedAmount}の${item}は、読む道具というより完全に趣味枠です。`
  ]
}
```

## 実装メモ

- 万能肯定文は出さず、商品と金額に寄せた短文を優先しています
- 同じ意味の重複や長すぎる文は、生成前に簡単なチェックで弾いています
- 旅館やグリーン車のような単発体験には、回数割りを安易に使わない方針です
