# 買い物言い訳ツール

商品名と金額を入力すると、その買い物をもっともらしく正当化する短い言い訳をブラウザ内だけで生成する、シンプルな静的Webアプリです。

くだらないけれど何度か試したくなる雰囲気を目指しており、商品カテゴリに応じて言い訳の方向性が変わるようにしています。

## サービス概要

- 商品名と金額を入力すると、日本語の短い言い訳を生成します
- 同じ入力でも「別の言い訳を考える」を押すと、なるべく違う文が出るようにしています
- 商品名に含まれる単語からカテゴリを判定し、カテゴリごとに使う論法を切り替えています
- 金額は日本円の3桁区切りで表示します
- コピー機能付きです

## 外部APIを使用していないこと

このアプリは以下の条件で動作します。

- 外部APIを使用しません
- OpenAI APIなどの生成AI APIを使用しません
- APIキーは不要です
- バックエンドを使用しません
- データベースを使用しません
- ログイン機能はありません
- 外部フォントやCDNを使用しません
- すべてブラウザ内だけで完結します

## ローカルでの起動方法

特別な準備は不要です。

1. このプロジェクトのファイル一式を手元に置きます
2. `index.html` をブラウザで開きます

ローカルサーバーなしでも基本機能は動作します。

## GitHub Pagesで公開できること

このプロジェクトは HTML / CSS / JavaScript だけで構成された静的サイトです。

そのため、GitHub Pages などの無料ホスティングでそのまま公開できます。

## 言い訳テンプレートを追加する方法

言い訳の文言は `script.js` の `categoryConfigs` にまとめています。

各カテゴリには以下の要素があります。

- `name`
  カテゴリ名
- `keywords`
  商品名に含まれていたらそのカテゴリとして扱う単語
- `intros`
  文の途中で使う短い説明文
- `templates`
  実際の言い訳テンプレート

### 追加手順

1. `script.js` の `categoryConfigs` を開く
2. 既存カテゴリの `intros` に文を追加する
3. 既存カテゴリの `templates` に新しいテンプレート関数を追加する
4. 新しい商品群を増やしたい場合は、新しいカテゴリを追加して `keywords` を設定する

### 例

```js
{
  name: "book",
  keywords: ["本", "書籍", "参考書"],
  intros: [
    "本は読んだ分だけちゃんと返ってきます。"
  ],
  templates: [
    ({ item, formattedAmount }) =>
      `${item}に${formattedAmount}を出したのは、その場で終わる買い物ではありません。読み切れば十分元は取れます。`
  ]
}
```

## 実装メモ

- カテゴリごとに使ってよい論法を分けています
- 旅館やグリーン車のように、回数割りが不自然なものには回数計算テンプレートを使わないようにしています
- スポンサー枠のような支援系は、損得より納得感を中心に組み立てています
